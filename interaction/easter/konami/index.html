KONAMI CODE

WHAT IS IT

The Konami Code is a cheat code that allows players to recieve special power-ups in many of Konami games. First developed by Kazuhisa Hashimoto, the code was a way for developers improve efficency during the debugging of the 1985 arcade game Gradius in the Nintendo Entertainment System (NES). The code was eventually popularized in North America through the NES version of Contra, providing an extra 30 lives when inputted.



The Easter Egg

Originally, this code was meant for only developers, yet was overlooked after consideration of possible game bugs after removal. The code was then popularized in many contemporary video games and media such as Tetris Effect, Fortnite, and Terraria.
